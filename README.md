## 📌 Project Overview

This repository provides a **step-by-step installation guide** for setting up a complete microcontroller development environment using Microchip tools.

It includes installation and configuration of essential tools required for **PIC microcontroller development and simulation**.

---

## 🧰 Tools Covered

This guide includes installation of:

- 🖥️ MPLAB X IDE (Development Environment)
- ⚙️ MPLAB XC8 Compiler (C Compiler for PIC Microcontrollers)
- 🧪 PICSimLab (Microcontroller Simulator)

---

## 🎯 Objective

The goal of this project is to help students and beginners:

- Set up a complete embedded development environment
- Understand required tools for PIC microcontroller programming
- Configure simulation tools for testing embedded code without hardware
- Build a strong foundation in Embedded Systems development

---

## 📚 What You Will Learn

By following this guide, you will learn:

- How to install MPLAB X IDE step-by-step
- How to configure XC8 compiler for PIC microcontrollers
- How to install and use PICSimLab simulator
- Basic setup required for embedded C programming workflow

---

## 🛠️ Installation Flow

1. Install MPLAB X IDE  
2. Install MPLAB XC8 Compiler  
3. Configure compiler with IDE  
4. Install PICSimLab simulator  
5. Verify setup for embedded development
